Miele per le nostre API

15.00 - 16.00, rooms.first

La proliferazione delle applicazioni mobile e di quelle web basate su javascript ci porta sempre più spesso a dover realizzare interfacce API. Ci sono tanti modi per farlo, e c'è il modo giusto. Illustreremo le best practice per la costruzione di un server API RESTful in RoR che lo rendano adatto ai nostri progetti, performante e manutenibile, vedendo insieme un esempio di loro implementazione.

Fabrizio Monti

Sviluppatore RoR per weLaika

Matteo Piotto

Ingegnere informatico, problem solver e appassionato di soluzioni e piattaforme open-source. Amante di caffeina, teina, gatti e fotografia, sviluppa applicazioni per Android e backend in PHP/RoR per weLaika.

Federico Parodi
