Web server challenge

11.00 - 12.00, rooms.last

Il dilemma della scelta del miglior web server per le nostre applicazioni è ricorrente nell'esperienza di uno sviluppatore Ruby/RoR. Approfondiremo meglio le caratteristiche uniche, i punti di contatto, ma soprattutto le divergenze tra i principali software utilizzati. Cercheremo di fornire una bussola con cui orientarsi e poter davvero comprendere come soddisfare l'esigenza specifica di ogni progetto.

Filippo Gangi Dino

Torinese, 31 anni, con più di dieci anni di esperienza nel­lo svi­lup­po web. Dopo un'esperienza legata prevalentemente al webdesign e al frontend, dal 2011 developer e socio in we­Lai­ka, so­cie­tà cooperativa in­for­ma­ti­ca torinese spe­cia­liz­za­ta in sviluppo di ap­pli­ca­zio­ni in Ru­by on Rails e sviluppo mo­bi­le iOS ed An­droid.

Alessandro Fazzi

Nato professionalmente sui sistemi unix, lavorando all'amministrazione di reti, mail server e soprattutto web server. Man mano convertito alla programmazione, ha aggiunto alle competenze di Bash scripting, Perl e PHP quelle di Javascript/Coffeescript, CSS/Sass. Apprendista stregone di Ruby on Rails. Developer e socio di weLaika.
